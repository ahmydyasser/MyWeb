PHP and MySQL

( Explaining Some PHP & MySQL )

PHP Notes

PHP Overview

  • PHP (Hypertext Preprocessor): A widely-used open-source scripting language, particularly suited for web development.
  • Server-Side: PHP code is executed on the server, and the result is sent to the client as plain HTML.
  • Syntax: PHP syntax is similar to C, Java, and Perl, making it easy to learn for people with basic programming skills.

Variables

  • Declaring Variables: Variables in PHP start with a $ sign, followed by the name of the variable.
  • Case Sensitivity: Variable names in PHP are case-sensitive.
  • Data Types: PHP automatically converts the variable to the correct data type, depending on its value.
  • Example: 
    $var1 = "Hello, World!";
                $var2 = 42;
                $var3 = 3.14;

Control Structures

  • If-Else Statements: 
    if (condition) {
                    // code to be executed if condition is true
                } else {
                    // code to be executed if condition is false
                }
  • Loops: PHP supports several types of loops such as for, foreach, while, and do-while. 
    for ($i = 0; $i < 10; $i++) {
                    echo $i;
                }

Functions

  • Defining Functions: 
    function functionName() {
                    // code to be executed
                }
  • Returning Values: Functions can return a value using the return statement. 
    function add($a, $b) {
                    return $a + $b;
                }
  • Parameters: Functions can accept parameters which are specified in the parentheses. 
    function greet($name) {
                    echo "Hello, " . $name;
                }

Superglobals

  • $_GET: Used to collect form data after submitting an HTML form with method="get". It also collects data sent in the URL. 
    // URL: example.com/index.php?name=John
                $name = $_GET['name'];
  • $_POST: Used to collect form data after submitting an HTML form with method="post". 
    $name = $_POST['name'];
  • $_SESSION: Used to store and retrieve session variables. 
    session_start();
                $_SESSION["user"] = "John";

MySQL Notes

MySQL Overview

  • MySQL: An open-source relational database management system (RDBMS) based on SQL (Structured Query Language).
  • Client-Server Architecture: MySQL follows a client-server architecture, where the server manages databases and the client requests services.
  • Data Storage: MySQL stores data in tables, which are organized into databases.

Database Operations

  • Creating a Database: 
    CREATE DATABASE mydatabase;
  • Selecting a Database: 
    USE mydatabase;
  • Dropping a Database: 
    DROP DATABASE mydatabase;

Table Operations

  • Creating a Table: 
    CREATE TABLE users (
                    id INT AUTO_INCREMENT PRIMARY KEY,
                    username VARCHAR(50) NOT NULL,
                    email VARCHAR(100)
                );
  • Inserting Data: 
    INSERT INTO users (username, email) 
                VALUES ('john_doe', 'john@example.com');
  • Updating Data: 
    UPDATE users 
                SET email = 'john_new@example.com' 
                WHERE username = 'john_doe';
  • Deleting Data: 
    DELETE FROM users 
                WHERE username = 'john_doe';

Queries

  • SELECT Query: Used to select data from a database. 
    SELECT * FROM users;
  • WHERE Clause: Used to filter records. 
    SELECT * FROM users 
                WHERE email = 'john@example.com';
  • JOIN Query: Used to combine rows from two or more tables. 
    SELECT users.username, orders.order_id 
                FROM users 
                JOIN orders ON users.id = orders.user_id;

Security Considerations

  • SQL Injection: Always validate and sanitize user input to prevent SQL injection attacks. 
    // Example of a vulnerable query:
                $unsafe_variable = $_GET["user_id"];
                $query = "SELECT * FROM users WHERE id = " . $unsafe_variable;
  • Password Hashing: Never store passwords as plain text. Use secure hashing algorithms like bcrypt. 
    $hashed_password = password_hash($password, PASSWORD_BCRYPT);
